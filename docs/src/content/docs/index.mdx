<section class="home-section">
  <p class="section-label">The Council</p>
  <h2>Different jobs deserve different agents.</h2>
  <p>Mjolnir turns a single coding session into a small, deliberate team. Each role has a clear responsibility, its own model strategy, and a place in the transcript.</p>
  <div class="role-grid">
    <article>
      <small>01 / COORDINATE</small>
      <h3>Thor directs</h3>
      <p>Owns the user turn, plans the work, and decides when a focused handoff will improve the result.</p>
      <code>role://thor</code>
    </article>
    <article>
      <small>02 / IMPLEMENT</small>
      <h3>Eitri forges</h3>
      <p>Receives bounded coding tasks in a fresh context, then streams the implementation back through the same UI.</p>
      <code>role://eitri</code>
    </article>
    <article>
      <small>03 / REVIEW</small>
      <h3>Loki challenges</h3>
      <p>Reviews checkpoints asynchronously and returns useful dissent at natural boundaries without stopping the work.</p>
      <code>role://loki</code>
    </article>
  </div>
</section>
